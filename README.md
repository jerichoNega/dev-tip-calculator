# Tip Pro - Advanced Tip Calculator

A modern, responsive React application built with Vite. This project demonstrates state management, component composition, and mobile-first design.

## Features

- **Dynamic Calculation**: Real-time tip and total updates.
- **Error Handling**: Visual validation for zero-person edge cases.
- **Preset Tips**: Quick selection buttons for common tip percentages.
- **Responsive UI**: Fully functional on mobile, tablet, and desktop.
